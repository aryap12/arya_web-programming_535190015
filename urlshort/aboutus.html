Tujuan



Web Shortener adalah sebuah web yang bertujuan untuk memudahkan sebuah user yang mempunyai sebuah link yang sebelumnya panjang dan susah untuk diingat menjadi lebih singkat dan mudah diingat. dengan memperpendek sebuah URL, maka URL tersebut semakin mudah untuk disebarkan dan diingat.

Aplikasi



Aplikasi yang kami gunakan dalam pembuatan Web Shortener ini adalah Visual Studio Code (VSC). alasan kami menggunakan VSC karena tampilan VSC yang user-friendly dan pada saat kami mengerjakan projek kami ini kebetulan kami sedang mengunakan Visual Studio Code (VSC)

Tanggal & Tempat



Projek ini kami kerjakan di rumah masing-masing karena kondisi pandemic yang tidak memungkinkan melakukan tatap muka. kami mengerjakan projek ini selama kurang lebih satu bulan yaitu dari pertengahan bulan September dan menyelesaikannya sampai pertengahan bulan Oktober.
